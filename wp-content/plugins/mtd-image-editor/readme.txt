Woocommerce extension for integrating the Aviary image editor into WooCommerce.

== Description ==
Woocommerce extension for integrating the Aviary image editor into WooCommerce. It allows customers to upload and edit their own images for ordering custom printed products.

== Installation ==
1. Upload \\\"mtd-image-editor\\\" directory to the \\\"/wp-content/plugins/\\\" directory.
2. Activate the plugin through the \\\"Plugins\\\" menu in WordPress.
3. Place \\\"do_action( \\\'plugin_name_hook\\\' );\\\" in your templates.
